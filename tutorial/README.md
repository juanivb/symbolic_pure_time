# Tutorials

- **`gsd_library/`** — start here. `gsd_quickstart.ipynb` (apply OLS & SPTLS:
  fit, summary, forecast, validation) and `gsd_diagnostics.ipynb` (the SPTLS
  reading suite).
- **`ols_vs_sptls/`** — univariate OLS-vs-SPTLS walkthrough, step by step, with
  a hand-checkable companion (`ols_vs_sptls_handcheck.pdf`).
- **`var_vs_sptls/`** — multivariate VAR-vs-SPTLS on cointegrated /
  non-stationary series.
