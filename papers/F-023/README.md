# Standard Asymptotic Theory Across Integration Orders via the Kinematic Jet

Running OLS on the kinematic jet q(t) = (y_t, Δy_t, Δ²y_t)' yields a
3×3 transition matrix estimator whose characteristic polynomial has
standard √T-Gaussian asymptotics uniformly across integration orders
d ∈ {0, 1, 2}. The central mechanism is a rank-one identity — the
estimation error has equal columns — which forces the Jacobian of the
characteristic polynomial map to be universal (the same matrix J
regardless of the data-generating process). This universality
annihilates the Brownian-motion contamination that ordinarily produces
non-standard limit theory under integration.

## Replication

All scripts depend only on NumPy. To reproduce every simulation:

```bash
pip install -r replication/requirements.txt

python replication/simulate_tables1_2.py
python replication/simulate_unit_root_asymptotics.py
python replication/simulate_garch_robustness.py
python replication/simulate_power_analysis.py
python replication/empirical_application.py
```

All scripts are deterministic (fixed seeds) and run end-to-end on
NumPy only. Expected runtime: ~5 minutes total (5000 replications
per script, T=500).

## Key results

The simulations confirm three claims made in the paper:

1. **√T-normality of char poly coefficients** (Tables 1–2): While individual
   elements of M̂ exhibit non-standard distributions under I(1), the
   characteristic polynomial coefficients (c₁, c₂, c₃) are √T-normal
   for all d ∈ {0, 1, 2}.

2. **Correct size under GARCH** (Table 4): The Newey-West HAC-based W₁*
   and W₂ tests maintain empirical size close to the nominal 5% level
   under GARCH(1,1) conditional heteroskedasticity (α=0.1, β=0.85).

3. **Complementary power** (Table 5): W₁* trades near-unit-root power
   for two capabilities ADF lacks: simultaneous coverage of d ∈ {0, 1, 2}
   and a companion W₂ test for I(2) vs I(1).
