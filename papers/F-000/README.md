# Symbolic Pure Time

Foundational paper of the programme. Introduces the substrate
$A \otimes \mathrm{Cl}(3,0)$ and Symbolic Pure-Time Least Squares (SPTLS):
the non-abelian one-step fit of the kinematic jet $q(t)=(z,\Delta z,\Delta^2
z)$, read by polar decomposition into a rotor (the genuinely non-abelian,
rotational content) and a stretch (the abelian gains an ordinary lag model
sees).

**Full text.** The PDF is withheld until the paper formally enters peer review; this folder provides the replication code and this summary in the meantime.
